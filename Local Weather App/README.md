#Local weather app

You can see the code running [Here](http://s.codepen.io/shah-0721/debug/mErJQv)

Created A Local Weathe App using HTML5, CSS3, Jquery ,Bootstrap

Used [Ip-api](http://ip-api.com/) to find the location of user Using Ip address

Worked With [Open Weather Api](http://openweathermap.org/api) to find the local weather also added additional features such as temperature convertion form c to F and F to c , image according to weather

Added additional feature for providing weather based upon location
